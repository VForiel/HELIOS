# 2025.11.25-02: Raccourcis d'imports pour API plus concise

## Résumé
Ajout de raccourcis d'imports dans `src/helios/__init__.py` pour rendre toutes les classes accessibles directement via `helios.Class` au lieu de `helios.components.Class` ou `helios.core.context.Class`.

## Objectif
Améliorer la lisibilité et réduire la verbosité du code utilisateur tout en conservant l'architecture modulaire existante.

## Modifications apportées

### Fichiers modifiés
- `src/helios/__init__.py` - Ajout des imports directs
- `build/lib/helios/__init__.py` - Synchronisation avec src
- `README.md` - Mise à jour de l'exemple avec la nouvelle syntaxe
- `examples/demo.ipynb` - Mise à jour de toutes les cellules avec la nouvelle syntaxe

### Changements
Ajout d'imports explicites de toutes les classes de composants et du core :

```python
# Expose core components
from .core.context import Context, Layer
from .core.simulation import Simulation

# Expose all component classes directly at package level for convenience
from .components import (
    # Scene components
    Scene, Star, Planet, ExoZodiacal, Zodiacal,
    # Optical components
    Collectors, Pupil, Coronagraph, BeamSplitter, FiberIn, FiberOut, Atmosphere, AdaptiveOptics,
    # Detector components
    Camera,
    # Photonic components
    PhotonicChip, TOPS, MMI
)

# Define public API
__all__ = [
    '__version__',
    'Context', 'Layer', 'Simulation',
    'components', 'core',
    'Scene', 'Star', 'Planet', 'ExoZodiacal', 'Zodiacal',
    'Collectors', 'Pupil', 'Coronagraph', 'BeamSplitter', 'FiberIn', 'FiberOut', 'Atmosphere', 'AdaptiveOptics',
    'Camera',
    'PhotonicChip', 'TOPS', 'MMI',
]
```

## Exemples d'utilisation

### Avant (syntaxe verbeuse)
```python
from helios.components import Star, Planet, Camera
from helios.core.context import Context
import astropy.units as u

star = Star(temperature=5778*u.K)
planet = Planet(position=(0.1*u.arcsec, 0.0*u.arcsec))
camera = Camera(pixels=(100, 100))
ctx = Context()
```

### Après (syntaxe concise)
```python
import helios
import astropy.units as u

star = helios.Star(temperature=5778*u.K)
planet = helios.Planet(position=(0.1*u.arcsec, 0.0*u.arcsec))
camera = helios.Camera(pixels=(100, 100))
ctx = helios.Context()
```

## Notes
Cette modification suit les bonnes pratiques Python de "flat is better than nested" tout en maintenant une structure de code source bien organisée. Les utilisateurs peuvent choisir le style qui leur convient :
- Import direct : `helios.Star()` (recommandé pour la simplicité)
- Import modulaire : `helios.components.Star()` (pour expliciter la catégorie)
- Import spécifique : `from helios import Star` (pour éviter les préfixes)
- Import wildcard : `from helios import *` (importe seulement les éléments de `__all__`)

L'ajout de `__all__` assure que les outils d'autocomplétion et de documentation (comme VS Code, Sphinx, etc.) connaissent l'API publique du package.
